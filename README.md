## 🌌 Опис проєкту
Цей інтерактивний освітній симулятор розроблений в межах дисципліни **"Інноваційні цифрові технології в освіті"**. Проєкт дозволяє візуалізувати складні астрофізичні процеси життєвого циклу зірок — від протозірки до білого карлика.

Проєкт поєднує в собі:
1. **3D/VR візуалізацію** на базі A-Frame.
2. **Науковий аналіз** за допомогою динамічної побудови діаграми Герцшпрунга-Рассела.
3. **Елементи штучного інтелекту** для класифікації спектральних класів зірок.

## 📂 Структура репозиторію
Згідно з рекомендаціями до виконання проєкту, структура організована наступним чином:

*   📂 `assets/` — Графічні ресурси та текстури.
*   📂 `docs/` — Детальна документація ([Посібник користувача](./docs/user-guide.md) та [Технічний опис](./docs/technical-docs.md)).
*   📂 `examples/` — Основна імерсивна сцена ([advanced-demo.html](./examples/advanced-demo.html)).
*   📂 `libs/` — Локальні копії зовнішніх бібліотек.
*   📂 `src/` — Вихідний код:
    *   `src/utils/` — Математичні розрахунки фізичних процесів.
    *   `src/ml/` — Модуль машинного навчання для класифікації.
*   📄 `index.html` — Головна точка входу з навігацією по проєкту.

## ✨ Ключові особливості
- **Інтерактивність:** Керування часом еволюції (від 0 до 15 млрд років) у реальному часі.
- **Наукова точність:** Зміна параметрів (L, T, R) відповідає реальним астрофізичним моделям.
- **VR-Ready:** Підтримка VR-окулярів та мобільних пристроїв.
- **AI-Integration:** Програмна імітація класифікатора спектрів на базі нейромережевих підходів.
