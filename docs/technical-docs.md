# Технічна документація
- **A-Frame:** Використовується для рендерингу 3D сцени.
- **Chart.js:** Використовується для візуалізації треку еволюції на діаграмі ГР.
- **Physics Module:** Розраховує радіус та світність на основі апроксимації даних зоряної еволюції.
- **ML Module:** Імітує роботу класифікатора спектрів на базі нейромережі.
